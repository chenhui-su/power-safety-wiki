## 🎯 如何认领主题

1. 待内容审核组确认事故清单并发布（以上清单仅为示例，具体以内容审核组发布为准）
2. 联系内容审核组登记认领，或自主选题并由内容审核组确认
3. 参考 [贡献指南](https://gitee.com/ch_su/power-safety-wiki/blob/main/CONTRIBUTING.md) 开始撰写

!!! info "认领须知"
    - 每人限认领 1 个主题
    - 请在规定时间内完成初稿
    - 如需更换主题，请提前告知内容审核组
