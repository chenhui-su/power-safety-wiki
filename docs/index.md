## 🎯 项目简介

电力系统是现代社会的命脉，大停电事故往往造成巨大的经济损失和社会影响。本案例库系统收集、整理全球重大停电事故，通过专业分析和 AI 辅助解读，为电力系统安全研究提供宝贵参考。

**我们的目标**：做电力系统安全分析方向最好的中文网站。

!!! info "当前阶段说明"
    当前仓库以「协作框架 + 内容模板 + 贡献指南」为主，部分案例页面为占位页，内容持续补充中。

## 🤖 AI 辅助分析

每个事故案例都配备了 AI 辅助分析视频，利用 NotebookLM 等先进工具，将复杂的技术报告转化为易于理解的对话式解读。

<div style="background: #f8f9fa; padding: 1.5rem; border-radius: 8px; margin: 1rem 0;">
  <strong>🎬 特色功能</strong>
  <ul>
    <li>基于官方调查报告的深度解读</li>
    <li>对话式呈现，通俗易懂</li>

## 🤝 参与贡献

我们欢迎所有对电力系统安全感兴趣的同学参与内容贡献！

!!! tip "如何参与"
    1. 📖 阅读 [贡献指南](https://gitee.com/ch_su/power-safety-wiki/blob/main/CONTRIBUTING.md)
    2. 🎯 认领感兴趣的事故主题
    3. 📝 按模板完成文档，并本地预览截图
    4. 🎬 使用 NotebookLM 生成分析视频
    5. 📦 打包提交内容组审核
    6. ✅ 技术组完成视频入库与页面部署

> **贡献者署名永久保留**，共同打造交大在电力安全分析领域的学术 IP。
