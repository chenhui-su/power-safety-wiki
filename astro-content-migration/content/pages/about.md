# 关于本项目

## 项目背景

电力系统是现代社会运转的关键基础设施。历史上的多次大规模停电事故表明，即使是技术先进、规模庞大的电网，也可能在复杂耦合条件下发生系统性故障。

本项目由上海交通大学电力系统安全分析相关课程与研究方向发起，目标是建设一个长期积累、可持续扩展的中文案例平台，系统整理全球重大停电事故，沉淀面向研究、教学与传播的内容资产。

## 项目目标

我们的目标是做电力系统安全分析方向高质量、可持续扩展的中文网站。

项目希望同时实现以下价值：

- 为课程教学和专业学习提供可检索的案例资料
- 将复杂事故转化为结构清晰、可复盘的中文内容
- 形成交大在电力系统安全分析方向的长期学术内容资产
- 让后续内容团队可以持续接力，而不是一次性项目

## 协作原则

- 内容为本：网站形式服务于内容表达
- 质量优先：关键结论必须有可靠来源支撑
- 结构统一：案例尽量遵循统一模板和字段规范
- 分工清晰：内容组负责写作，技术组负责接入与发布

## 团队分工

### 组织组

负责任务协调、进度推进和协作安排。

### 内容审核组

负责主题规划、写作规范、质量把关和审核反馈。

### 技术组

负责站点搭建、内容接入、视频嵌入和部署发布。

### 内容策划组

负责事故案例写作、资料整理、视频制作与提审。

## 当前阶段说明

当前阶段已经完成迁移内容框架的整理，但尚未正式迁入案例正文。后续工作重点是：

1. 建立 Astro 新项目骨架
2. 接入 Markdown 内容集合
3. 先迁入项目说明和模板
4. 逐步补齐首批正式事故案例
